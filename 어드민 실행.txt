🔍 이유

제가 만들어드린 버전의 auth_manager.py는
Google OAuth 연동 전에 기본적으로 이렇게 설정되어 있어요:

USERS = {
    "admin@seoul.com": {
        "name": "서울 관리자",
        "password": "password123",
        "branch_id": "seoul_law"
    }
}


Streamlit 실행 시 로그인 창이 뜨고,
email = admin@seoul.com, password = password123
로 입력하면 정상적으로 들어갈 수 있습니다.

① 테스트용 로그인

그냥 이렇게 입력해보세요 👇

이메일: admin@seoul.com

비밀번호: password123

맨 위 부분을 이렇게 고치세요 👇

USERS = {
    "윤찬@hanwoory.com": {    # 또는 Gmail 주소
        "name": "윤찬 행정사",

        "branch_id": "seoul_law"
    }
